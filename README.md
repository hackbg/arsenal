# Arsenal ⚔️💣☣️

Unified shell platform based on Nix.

## Usage

### Installation

```bash
git clone git@github.com:hackbg/arsenal.git
arsenal/install
```

### Standalone usage

The `install` script will add the `arsenal` alias to your `~/.profile`.
Run it to launch a shell with the tools contained in Arsenal:

```bash
arsenal
```
